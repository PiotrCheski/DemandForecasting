The Demand Forecasting web application is an analytical tool that allows the analysis of transaction history from an online store. The Demand Forecaster enables the visualization of sales results for selected products in a chosen time period. The web application allows demand forecasting using four methods:

  • Constant average method
  • Moving averages method
  • Exponential smoothing method
  • Linear regression method
